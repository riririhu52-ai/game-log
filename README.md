# 🎮 Multiplication Game (Python)

## 📌 Overview
This is an interactive multiplayer multiplication game built using Python. Players compete by answering multiplication questions under different game modes such as speed rounds, bonus rounds, and sudden death challenges.

The game is designed to improve mental math skills while making learning fun and competitive.

---

## ⚙️ Features
- 👥 Multiplayer support
- ⏱️ Timed questions (speed challenge)
- 🔥 Random special rounds (double points, bonus, speed rounds)
- 🎯 Streak bonus system
- 💀 Sudden death round
- 👑 Final boss question
- 🏆 Live scoreboard tracking

---

## 🧠 How It Works
1. Teacher enters name and number of players
2. Each player answers multiplication questions
3. Game randomly assigns special rounds with different rules
4. Scores are tracked throughout all rounds
5. Final winner is announced at the end
